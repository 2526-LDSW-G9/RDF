This data story explores a linked dataset of wildlife occurrence observations, combining local biodiversity records with taxonomic knowledge from Wikidata. Each query below is live — results are fetched directly from the triple store.

The dataset uses the Darwin Core vocabulary (dwc:) for occurrence records, and GeoSPARQL for geometries — enabling spatial queries over observation locations. Taxonomic lineage is resolved at query time via a federated SERVICE call to Wikidata.

Occurrences with Geometry — Federated via Wikidata

This query retrieves up to 300 wildlife occurrence records, each with a Dutch vernacular name, an observation date, and a GeoJSON geometry. It federates with Wikidata to resolve the taxon's scientific name — linking local records to the global taxonomic graph. Results cover taxa descending from Q176901 (Squamata — scaled reptiles).

After running the query, switch to the 🗺 Map tab to see all observations plotted on an OpenStreetMap base layer. Click any marker to see the species name.
